
# Finding even number

An app to check if number is prime or not . 


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

move the pdf file with name sample.pdf 


Start the server

```bash
  npm run dev
```
